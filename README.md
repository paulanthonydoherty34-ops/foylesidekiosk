HOW TO LAUNCH
─────────────
WINDOWS:
  Double-click LAUNCH_WINDOWS.bat
  Chrome will open in full kiosk mode automatically.
  Press Alt+F4 or Ctrl+W to exit kiosk mode.

MAC:
  Double-click LAUNCH_MAC.command
  If blocked by security: right-click > Open > Open

XIAOHUI SCREEN / DIGITAL SIGNAGE:
  Use the HTML layer in the xiaohui editor.
  Either upload index.html to a web server and paste
  the URL into the route field, OR check if your
  xiaohui player supports loading from USB/local path:
  file:///[USB drive letter]/index.html

MANUAL OPEN (any computer):
  Open index.html in Chrome or Edge.
  Press F11 for fullscreen.


REQUIREMENTS
────────────
  - Google Chrome (recommended) or Microsoft Edge
  - No internet connection required
  - Screen resolution: 2160 × 3840 (9:16 portrait)
    (also works on any screen in scaled mode)


EDITING THE DIRECTORY
─────────────────────
  Use the foyleside_editor.html file (separate download)
  to add/edit/remove stores, then click Export to
  generate an updated index.html to replace this one.


FEATURES
